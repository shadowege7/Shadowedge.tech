# ShadowEdge Managed IT Services Website

A professional, SEO-friendly static marketing website for ShadowEdge Managed IT Services.

## Features
- Mobile-first responsive design
- Semantic HTML5 structure
- Strong SEO foundation with title tags, meta descriptions, robots.txt, and sitemap.xml
- Service and location landing pages
- Contact page with a form placeholder ready for integration

## Project structure
- index.html — homepage
- services.html — services overview
- about.html — company overview
- locations.html — location overview
- resources.html — resources/blog landing page
- contact.html — contact page
- location-*.html — location-specific landing pages
- service-*.html — individual service pages
- assets/ — CSS and JavaScript

## Preview locally
Open index.html in a browser, or serve the folder with a simple local server.
