1. GitHub access token

Publishing works by committing files directly to the shadowege7/Shadowedge.tech GitHub repo, which triggers the existing auto-deploy. Create a fine-grained personal access token at github.com/settings/personal-access-tokens/new scoped to only this repository, with Contents: Read and write permission (nothing else needed). The token is stored only in this browser (localStorage) and is never committed anywhere — treat it like a password and don't share this page or the token with anyone you don't trust with write access to the site.
